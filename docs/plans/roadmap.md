# Roadmap (MVP)

## MVP Scope
The initial version of the system will include:
- Create a task
- Retrieve all tasks
- Update a task
- Delete a task

Data will be stored in-memory.

---

## Stage 1: Project Setup
- Define project structure
- Implement basic models (Task case class)
- Set up repository layer (in-memory storage)

---

## Stage 2: Core Functionality
- Implement service layer logic
- Add CRUD operations
- Ensure proper separation of concerns

---

## Stage 3: API Layer
- Implement REST endpoints (routes)
- Connect routes to service layer
- Handle request/response mapping

---

## Stage 4: Improvements
- Add validation
- Improve error handling
- Refactor code if needed

---

## Future Enhancements
- Integrate database (e.g., PostgreSQL)
- Add authentication and authorization
- Implement filtering and pagination
