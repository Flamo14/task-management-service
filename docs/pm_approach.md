# Project Management Approach

This project follows a hybrid approach combining elements of Kanban and iterative development. Work is organized into small, incremental tasks that can be implemented and tested independently. The focus is on continuous progress and maintaining a working system at all times.

Responsibilities are divided between the developer and the AI assistant. The developer is responsible for architectural decisions, system design, and code review. The AI assistant is used to generate boilerplate code, suggest implementations, and assist with writing unit tests. All AI-generated code must be reviewed and validated before integration.
